## 🧩 Python Dashboard Demo

פרויקט קטן לדוגמה ב‑Python שמדגים פונקציית "דשבורד" פשוטה בקובץ `dashboard.py`.  
הקוד מציג נתוני משתמש (משימות שהושלמו, הודעות, התראות ותאריך התחברות אחרון) בצורה טקסטואלית.

### 📂 מבנה הפרויקט

```text
github2/
├─ dashboard.py   # דשבורד טקסטואלי לדוגמה
└─ README.md      # תיעוד הפרויקט
```

### 🚀 הפעלה מהירה

1. ודא ש‑Python מותקן במחשב (מומלץ ‎3.8+‎).
2. פתח טרמינל בתיקיית הפרויקט (`github2`).
3. הרץ:

```bash
python dashboard.py
```

ברירת המחדל מריצה דשבורד למשתמש בשם `Alice`.

### 🧪 שימוש בפונקציה מהקוד

ניתן לייבא את הפונקציות ישירות בקוד Python אחר:

```python
from dashboard import show_dashboard

show_dashboard("Bob")
```

הפלט יכלול את ברכת הדשבורד ונתוני המשתמש לדוגמה.

### 🧠 פונקציות עיקריות

- **`show_dashboard(user)`**: מקבלת שם משתמש ומציגה דשבורד טקסטואלי.
- **`get_dashboard_data(user)`**: מחזירה מילון (dict) עם נתוני דשבורד לדוגמה.

### 📌 רעיונות להמשך פיתוח

- **חיבור למקור נתונים אמיתי** (API / מסד נתונים) במקום נתונים קשיחים.
- **המרה לממשק גרפי** (GUI) או לאפליקציית ווב.
- **הוספת בדיקות יחידה** לפונקציות.
- **קבלת שם משתמש מהקלט** (קומנד ליין / ממשק משתמש).
